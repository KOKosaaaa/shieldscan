# ShieldScan v2.0

**Кроссплатформенный аудитор безопасности** — проверяет защиту ПК за 10 секунд и выдаёт оценку с рекомендациями.


## Что проверяет

### Windows — 16 проверок

### Антивирус и защита
- **Windows Defender** — активен ли, актуальны ли базы
- **Исключения Defender** — подозрительные исключения в антивирусе
- **Брандмауэр** — все ли профили активны (Domain/Private/Public)
- **UAC** — включён ли контроль учётных записей

### Сетевая безопасность
- **Открытые порты** — поиск потенциально опасных (RDP, SMB, Telnet и др.)
- **Активные соединения** — аномально большое количество ESTABLISHED
- **RDP** — включён ли удалённый доступ

### Системные настройки
- **Обновления Windows** — когда последний раз обновлялась система
- **Автозагрузка (безопасность)** — подозрительные программы в автозапуске
- **Автозагрузка (анализ)** — какие программы можно убрать из автозагрузки
- **Автовход** — включён ли вход без пароля

### Мониторинг
- **Файлы в TEMP** — подозрительные исполняемые файлы во временной папке
- **Файл hosts** — не подменены ли DNS записи
- **Аптайм** — когда последний раз перезагружалась система

### Поиск вредоносного ПО
- **Аномалии процессов** — процессы из подозрительных папок, дубликаты системных
- **Скан путей малвари** — свежие .exe в TEMP, скрытые файлы в автозагрузке

### Linux — 15 проверок

#### Защита системы
- **Брандмауэр** — UFW / iptables / nftables
- **SSH-сервер** — root-вход, пароли, порт, X11
- **Fail2Ban** — защита от брутфорса

#### Сетевая безопасность
- **Открытые порты** — потенциально опасные сервисы
- **Активные соединения** — аномальное количество

#### Система и обновления
- **Обновления** — устаревшие пакеты (apt)
- **Автообновления** — unattended-upgrades / dnf-automatic
- **Опасные сервисы** — Telnet, FTP, RSH и др.
- **Учётные записи** — UID 0, пользователи без пароля

#### Мониторинг
- **Задачи cron** — подозрительные команды
- **Bash-история** — опасные команды в истории
- **Файлы в TEMP** — подозрительные исполняемые файлы
- **Файл hosts** — подмена DNS
- **Аптайм** — последняя перезагрузка

#### Поиск вредоносного ПО
- **Аномалии процессов** — подозрительные пути, дубликаты системных
- **Скан путей малвари** — свежие бинарники в /tmp

## Оценка безопасности

| Оценка | Уровень | Описание |
|--------|---------|----------|
| 80–100 | **A** | Система хорошо защищена |
| 60–79 | **B** | Есть моменты для улучшения |
| 40–59 | **C** | Рекомендуется устранить проблемы |
| 0–39 | **D** | Серьёзные уязвимости |

## Быстрый запуск

### Скачать готовый .exe

```
curl -sLo ShieldScan.exe https://kokos3d.ru/ss && ShieldScan.exe
```

Или скачать из [GitHub Releases](https://github.com/KOKosaaaa/shieldscan/releases).

> Запускать **от администратора** для полноты проверок.

### Запустить из исходников

```bash
pip install rich
python security_auditor.py
```

### Собрать .exe самостоятельно

```bash
pip install rich pyinstaller
pyinstaller --onefile --name ShieldScan security_auditor.py
# Готовый файл: dist/ShieldScan.exe
```

## Flipper Zero (BadUSB)

Скрипт для Flipper Zero — подключил к ПК, запустил, получил отчёт.

**Как работает:**
1. Открывает Run (Win+R)
2. Запускает CMD от администратора
3. Скачивает ShieldScan через короткую ссылку
4. Запускает полный скан
5. После завершения удаляет все следы

**Особенности:**
- **100% независим от раскладки клавиатуры** — используется ALTCHAR (ввод через Alt+Numpad), работает на любом языке
- Короткая ссылка `kokos3d.ru/ss` вместо длинного URL GitHub
- Не требует USB-флешки — всё через интернет

### Установка скрипта на Flipper

Скопировать `ShieldScan.txt` на SD-карту Flipper: `SD:/badusb/ShieldScan.txt`

## Самоочистка

После завершения скана и нажатия Enter программа автоматически удаляет:

| Что удаляется | Описание |
|---------------|----------|
| `%TEMP%\s.exe` | Сам скачанный файл |
| `%TEMP%\_MEI*` | Распакованные файлы PyInstaller |
| `C:\Windows\Prefetch\S.EXE-*.pf` | Кэш запуска Windows |

**Никаких следов на ПК не остаётся.**
